# EUNOIA

> Responsive beauty & cosmetics eCommerce website with clean and elegant design
> 
#### BUILT WITH

* HTML5
* CSS3
* React
* React Router 6
* JavaScript
* API (Airtable)
* Auth0

#### DEMO

https://eunoia-store.netlify.app/

#### FEATURES

* responsive
* featured products
* product sorting and filtering
* product page column and row layout
* shopping cart
* newsletter signup form
* login
